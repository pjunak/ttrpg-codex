Third-party assets bundled with this repository, with their authors and licenses.

## Bundled marker icons (`web/icons-defaults/`)

The default map-marker SVGs in [`web/icons-defaults/`](web/icons-defaults/) are sourced from [**game-icons.net**](https://game-icons.net/) and are licensed under the **Creative Commons Attribution 3.0 Unported** license ([CC BY 3.0](https://creativecommons.org/licenses/by/3.0/)).

**Modification notice:** the original SVGs ship with an opaque black background path (`<path d="M0 0h512v512H0z"/>`); each file in this folder has had that background removed so the foreground silhouette is transparent — the colored attitude-glow filter on map markers can then trace the icon outline rather than the bounding box. No other changes were made.
